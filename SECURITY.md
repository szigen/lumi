# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Lumi, please report it responsibly.

**Do not open a public issue.** Instead, email: **szigen@pm.me**

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a fix promptly.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
